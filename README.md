## beginning
```bash
npm init -y
npm i -D typescript
npx tsc --init
npm i -D @types/node @types/jest jest prettier ts-jest tslint tslint-config-prettier

npm i --save-peer acorn acorn-walk fast-xml-parser quickjs-emscripten
```

## publish
```bash
npm version patch
```
